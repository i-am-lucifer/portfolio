Summarization of Lifelog Data Using LLMs

Developed a state-of-the-art solution leveraging large language models (LLMs) to summarize lifelog data for users, enabling easy extraction of meaningful insights from complex data logs.

Problem Statement

Lifelog data, collected from various devices, contains large volumes of unstructured information. Users struggled to extract actionable insights efficiently.

Approach

  • Preprocessed lifelog data using Python and NLP libraries for structured input.
  • Integrated OpenAI's GPT model for summarization and fine-tuned it for domain-specific tasks.
  • Designed a user-friendly interface to deliver concise daily and weekly summaries.

Results

The solution reduced manual effort by 80%, providing actionable insights to users in seconds. Achieved a BLEU score of 0.85 for summarization quality.
