Predicting Target Customers

This project focuses on predicting customers likely to buy electronics using a retail dataset. The goal is to identify potential customers and optimize marketing strategies based on data-driven insights.

Problem Statement

Businesses often struggle to identify target customers due to the lack of actionable insights from vast amounts of transactional and behavioral data. By leveraging machine learning techniques, this project aims to solve this problem efficiently.

Approach

  • Data Cleaning: Processed and cleaned a dataset of 1 million rows, including handling missing values and nested columns.
  • Feature Engineering: Created meaningful features like engagement score, recency, and category frequencies to improve prediction accuracy.
  • Class Balancing: Addressed class imbalance using synthetic data generation techniques.
  • Modeling: Applied Logistic Regression, Random Forest, and XGBoost to predict customer behavior.
  • Evaluation: Used accuracy, precision, recall, and F1-score to assess model performance.

Results

The Logistic Regression model achieved a balanced performance with an accuracy of 67.9%, while Random Forest and XGBoost models performed exceptionally well on synthetic data but risked overfitting. Feature importance analysis highlighted total spend, engagement score, and annual income as the most influential factors.
