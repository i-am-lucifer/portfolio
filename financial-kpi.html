Apple Stock Financial KPI Dashboard

This project showcases a Financial KPI Dashboard developed using Tableau Public to analyze Apple stock performance from 2019 to 2024. The dashboard includes insights derived from stock prices, moving averages, Bollinger Bands, volatility, and cumulative returns.

Problem Statement

Financial analysts need an interactive and visually appealing way to track and evaluate the performance of stocks over time. This dashboard aims to provide actionable insights into stock trends and metrics, facilitating informed decision-making.

Approach

  • Data Collection: Downloaded historical stock data for Apple from Yahoo Finance using Python and the yfinance library.
  • Data Preparation: Processed data to calculate metrics such as moving averages, Bollinger Bands, daily returns, and cumulative returns.
  • Visualization: Designed an interactive Tableau dashboard with annotations, tooltips, and filters for enhanced user experience.
  • Insights: Highlighted key events, such as the impact of the COVID-19 pandemic on stock performance.

Results

The dashboard provides a comprehensive view of Apple stock performance over five years, enabling users to:

  • Track stock trends with moving averages.
  • Identify periods of high volatility.
  • Analyze cumulative growth and returns over time.
  • Gain insights into market behavior through Bollinger Bands and daily return distributions.
